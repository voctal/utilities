## About

Useful TypeScript utilities built to power our own projects, and yours too. We plan to ship new utilities regularly as we continue to grow our stack.

## Packages

- [@voctal/cache](https://github.com/voctal/utilities/tree/main/packages/cache) - Lightweight and performant caching library
- [@voctal/ms](https://github.com/voctal/utilities/tree/main/packages/ms) - Convert various time formats to milliseconds. Enhanced and localized version of vercel/ms
- [@voctal/snowflake](https://github.com/voctal/utilities/tree/main/packages/snowflake) - Utility package to generate and deconstruct snowflakes
- [@voctal/duration](https://github.com/voctal/utilities/tree/main/packages/duration) - Time and duration utility package
